Step 1: 申請書転記データ生成

申請書_転記データシートに当月分の集計データを書き込みます（upsert・冪等）。
申請書PDF生成の前に必ず実行してください。 要確認フラグがある場合は先にスプレッドシートで確認してください。

Step 2: 申請書 NDJSON 出力（推奨）

申請書データを NDJSON 形式で Drive に出力します。
ダウンロード後、ローカルで python write_application.py --batch <ファイル名> を実行すると
施術日カレンダー○・転帰・金額が正しく入った Excel 申請書が生成されます。
この方式が帳票品質として正確です（施術日カレンダーの○印・転帰処理を含む）。

Step 3: 申請書Excelを生成（B案 Cloud Run・正ルート）

既存B案 Cloud Run ルートで Excel 申請書を生成します。
施術日カレンダーの○・転帰・金額が正しく入った帳票品質の申請書が生成されます。
生成ファイルは Drive の月別フォルダに保存され、申請書生成ログに記録されます。
請求確定前に内容を確認してください。
⚠️ 要確認フラグがある来院があります。申請書生成前にスプレッドシートで内容を確認してください。

Step 4: 申請書PDF直接生成（試験的 / 帳票品質確認中）

⚠️ Sheets直PDF方式は施術日カレンダーの○印・転帰処理が未実装です。
帳票として正確な出力には Step 2 の NDJSON + Python スクリプトをお使いください。
⚠️ 要確認フラグがある来院があります。PDF を生成しても、月次申請前に Sheets で確認が必要です。
